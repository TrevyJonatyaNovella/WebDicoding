About Bangkit

Designed to prepare students with in-demand skills and tech certifications, the new Bangkit curriculum offers 3 interdisciplinary learning paths machine learning, mobile development, and cloud computing. By the end of this program, you'll be equipped with the tech expertise and soft skills you need to transit from academia to the workplace and succeed at leading companies.

Learn from and connect with tech experts from leading companies in Indonesia

Path Learning

Bangkit curriculum offers 3 interdisciplinary learning paths machine learning, mobile development, and cloud computing.

Machine Learning

Learn the key concepts and applications of AI to solve a wide range of ML problems with these specializations:

  • Google IT Automation with Python
  • DeepLearning.AI TensorFlow Developer Professional Certificate program
  • TensorFlow: Data and Deployment Specialization

In this path, you'll prepare for the TensorFlow Developer Certificate exams.

Mobile development

Learn the fundamental concepts and core skills to launch your career as a professional Android developer.
Courses are adapted from Google's Android Developer Fundamentals and Advanced courses and offered through Dicoding.
In this path, you'll prepare for the Google Associate Android Developer Certification exam.

Cloud computing

Learn the fundamentals of cloud computing to deploy applications, monitor operations, and manage enterprise solutions with these specializations:

  • Google IT Automation with Python
  • Google Cloud Computing Foundations
  • Architecting with Google Compute Engine
  • From Data to Insights with Google Cloud Platform

In this path, you'll prepare for the Associate Cloud Engineer Certification exam.
